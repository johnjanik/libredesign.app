# DesignLibre Menu Structure

## File Menu

| Menu Item | Shortcut | Submenu | Description |
|-----------|----------|---------|-------------|
| New Design | Ctrl+N | - | Create a new blank design |
| Open... | Ctrl+O | - | Open existing file |
| Save | Ctrl+S | - | Save current document |
| Save As... | Ctrl+Shift+S | - | Save with new name |
| Export | - | > PNG | Export as PNG |
| | | > SVG | Export as SVG |
| | | > PDF | Export as PDF |
| Close | Ctrl+W | - | Close current document |

## Edit Menu

| Menu Item | Shortcut | Submenu | Description |
|-----------|----------|---------|-------------|
| Undo | Ctrl+Z | - | Undo last action |
| Redo | Ctrl+Shift+Z | - | Redo last undone action |
| Cut | Ctrl+X | - | Cut selection |
| Copy | Ctrl+C | - | Copy selection |
| Paste | Ctrl+V | - | Paste from clipboard |
| Copy As | - | > CSS | Copy as CSS code |
| | | > SVG | Copy as SVG markup |
| | | > PNG | Copy as PNG image |

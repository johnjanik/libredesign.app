## Platform Notes

- On macOS, use `Cmd` instead of `Ctrl` for most shortcuts
- The `Meta` key (Windows key on Windows, Cmd on Mac) works as an alternative to `Ctrl`
- Some shortcuts may be overridden by browser or OS-level shortcuts

## Tips

1. **Quick tool switching**: Press a tool's key to switch to it, press `V` to return to the select tool
2. **Precise positioning**: Hold `Shift` while nudging for 10px increments
3. **Zoom to point**: Use `Ctrl + Mouse Wheel` to zoom centered on cursor position
4. **Copy properties**: Use `Ctrl+Alt+C` to copy style properties, then `Ctrl+Alt+V` to apply to other objects

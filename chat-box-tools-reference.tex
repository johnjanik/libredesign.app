\begin{abstract}
This document provides two comprehensive lists: (1) tools that can be implemented in a design tool chat box based on existing Plugin API capabilities and native features, and (2) tools and features that users have explicitly requested through forums, wishlists, and community feedback. These lists inform the development of an AI-powered chat assistant for design applications.
\end{abstract}

%==============================================================================
\section{Overview}
%==============================================================================

This reference document is organized into two major sections:

\begin{itemize}
  \item \textbf{Part I: Available Tools} --- Tools implementable today via Plugin API, REST API, and native features
  \item \textbf{Part II: User-Requested Tools} --- Features users explicitly want, gathered from forums, wishlists, and community feedback
\end{itemize}

Each tool entry includes:
\begin{itemize}[noitemsep]
  \item Tool name and description
  \item Category classification
  \item Example invocation phrase
  \item Technical implementation notes (where applicable)
\end{itemize}
